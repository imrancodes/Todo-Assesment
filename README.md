### Installation

To get a local copy up and running, follow these steps:

1. **Clone the repository:**

```bash
git clone https://github.com/imrancodes/Todo-Assesment
```

2. **Navigate to the project directory:**

```bash
cd Todo-Assesment
```

3. **Install Dependencies**

```bash
npm install
```
4. **Run the Project** 
```bash
npm run dev
```
